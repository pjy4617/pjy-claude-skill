## 아키텍처 개요

```
PHY IC (DP83848 등)         트랜스포머            RJ45
┌──────────┐              ┌──────────┐        ┌──────┐
│  TX+ (MDI)├──33R──┬─────┤ 1:1      ├──ESD──►│ Pin 1│──→ 네트워크
│  TX- (MDI)├──33R──┘     │ 100Base-TX│        │ Pin 2│
│           │              │          ├──ESD──►│      │
│  RX+ (MDI)├──────┬──────┤ 1:1      │        │ Pin 3│
│  RX- (MDI)├──────┘      │          │        │ Pin 6│
└──────────┘              └────┬─────┘        └──┬───┘
                               │ CT               │
                          Bob Smith 종단      Shield GND
                          (75Ω + 1000pF)
```

## 펄스 트랜스포머

### 역할

- DC 절연 (갈바닉 아이솔레이션) — PHY와 네트워크 장비 간 전기적 분리
- 임피던스 매칭 (100Ω 차동)
- 커먼모드 노이즈 제거

### 선정 기준

| 항목 | 100Base-TX 요구 | 비고 |
|------|----------------|------|
| 권선비 | 1:1 | 10/100Mbps 표준 |
| 절연 전압 | ≥1500Vrms | IEEE 802.3 요구 |
| 삽입 손실 | ≤1.1dB @ 80MHz | 신호 품질 |
| 리턴 손실 | ≥16dB | 반사 최소화 |
| OCL (개방 회로 인덕턴스) | ≥350μH | 저주파 전송 |

### 대표 부품

| 파트넘버 | 제조사 | 포트 수 | 패키지 | 비고 |
|---------|--------|---------|--------|------|
| HX1188NL | Pulse Electronics | 1 | SMD-16 | 가장 범용적 |
| H1102NL | Pulse Electronics | 1 | SMD-16 | HX1188 대체 |
| ST8024S | YCL | 1 | SMD-16 | 저가형 |
| HR601680 | Hanrun | 1 | SMD-16 | 중국 로컬 |

### 연결 방법

```
PHY TX+ ──[33Ω]──┬── 트랜스포머 1차 TX+ (Pin 1)
PHY TX- ──[33Ω]──┘── 트랜스포머 1차 TX- (Pin 3)
                      트랜스포머 1차 CT  (Pin 2) → +3.3V (100nF 바이패스)

트랜스포머 2차 TX+ (Pin 6) ── RJ45 Pin 1
트랜스포머 2차 TX- (Pin 4) ── RJ45 Pin 2
트랜스포머 2차 CT  (Pin 5) → Bob Smith 종단
```

### 주의사항

- 센터탭(CT)은 PHY 측은 VCC(+3.3V)에, RJ45 측은 Bob Smith 종단에 연결
- PHY 데이터시트의 Application Circuit에서 CT 연결 방법이 지정되어 있으면 반드시 따른다
- 일부 PHY(예: DP83848)는 내부 종단이 있어 외부 직렬 저항이 불필요할 수 있음 — 데이터시트 확인 필수

## RJ45 커넥터

### 유형

| 유형 | 설명 | 장점 | 단점 |

| 내장 마그네틱스 | RJ45 + 트랜스포머 일체형 | 면적 절감, 설계 간편 | 부품 단가 높음 |

## ESD TVS 보호

### 역할

외부 커넥터(RJ45)에 연결되는 MDI 라인은 ESD에 취약하다. TVS(Transient Voltage Suppressor) 다이오드로 보호한다.

### 선정 기준

| 항목 | 권장값 | 비고 |
|------|--------|------|
| 작동 전압 (Vrwm) | ≥5V | 신호 전압 이상 |
| 클램핑 전압 | ≤30V @ 8/20μs | PHY 허용 범위 내 |
| 캐패시턴스 | ≤5pF | 신호 왜곡 최소화 (100Base-TX) |
| ESD 등급 | ≥8kV (접촉), ≥15kV (공기) | IEC 61000-4-2 |

### 대표 부품

| 파트넘버 | 제조사 | 채널 | 패키지 | Cj |
|---------|--------|------|--------|-----|
| CDSOT23-SM712 | Bourns | 2 | SOT-23-3 | 3.5pF |
| CDSOD323-T05C | Bourns | 2 | SOD-323 | 5pF |
| PESD5V0S2BT | Nexperia | 2 | SOT-23 | 3pF |
| SRV05-4 | Littelfuse | 4 | SOT-23-6 | 0.65pF |

### 배치 위치

- 트랜스포머 2차 측 (RJ45 쪽)에 배치
- TX± 페어와 RX± 페어에 각각 1개씩 (총 2개/포트)
- 접지는 쉴드 GND (섀시 GND) 에 연결 — 신호 GND와 분리

## Bob Smith 종단

### 역할

트랜스포머 2차 측 센터탭에 연결하여 커먼모드 노이즈를 접지로 바이패스한다.
IEEE 802.3 규격에서 권장하는 표준 종단 방식이다.

### 회로

```
트랜스포머 2차 CT (TX) ──[75Ω]──┬──[1000pF/2kV]── 섀시 GND
                                │
트랜스포머 2차 CT (RX) ──[75Ω]──┘
```

### 부품 사양

| 부품 | 값 | 정격 | 비고 |
|------|-----|------|------|
| 저항 | 75Ω (±1%) | 1/8W 이상 | 0402 또는 0603 |
| 캐패시터 | 1000pF (1nF) | 2kV 이상 | 절연 확보, 1206 또는 0805 |

### 주의사항

- 캐패시터의 정격 전압이 중요하다 — 트랜스포머 절연 전압과 맞춰야 함
- 섀시 GND(Shield)와 신호 GND는 1nF/2kV 캡으로 AC 커플링하는 것이 일반적
- 일부 설계에서는 75Ω 대신 49.9Ω을 사용 — PHY 데이터시트의 권장 회로 참조

## 커먼모드 초크 (CMC)

### 역할

PHY TX/RX 라인의 커먼모드 노이즈를 억제한다. EMC 인증에 유리하지만, 100Base-TX에서는 필수가 아닌 경우가 많다.

### 적용 조건

| 상황 | CMC 필요 여부 |
|------|-------------|
| 100Base-TX, 짧은 케이블 (<10m) | 보통 불필요 |
| EMC 인증 필요 (CE, FCC) | 권장 |
| 1000Base-T (기가비트) | 필수 |
| 긴 케이블 (>30m), 노이즈 환경 | 권장 |

### 대표 부품

| 파트넘버 | 제조사 | 임피던스 | 패키지 |
|---------|--------|---------|--------|
| ACM2012-900-2P | TDK | 90Ω@100MHz | 0805 |
| DLW21SN900SQ2 | Murata | 90Ω@100MHz | 0805 |
| WE-CNSW 744232090 | Würth | 90Ω@100MHz | 0805 |

### 배치 위치

- PHY TX/RX 출력과 트랜스포머 1차 사이
- 직렬 저항 앞 또는 뒤 (PHY 데이터시트 참조)

## 체크리스트

Ethernet 프런트엔드 리뷰 시 적용하는 검증 항목. `review-checklist.md`의 F(도메인별) 카테고리에서 Ethernet PHY 감지 시 사용한다.

| ID | 항목 | ERROR 조건 | 검증 방법 |
|----|------|-----------|----------|
| ETH1 | 펄스 트랜스포머가 MDI와 RJ45 사이에 배치되었는가 | 트랜스포머 없음 → DC 절연 없음 | 트랜스포머 부품 존재 확인 |
| ETH2 | 트랜스포머 권선비가 1:1인가 (100Base-TX) | WARN: 권선비 불일치 | 데이터시트 확인 |
| ETH3 | 트랜스포머 센터탭(CT)이 올바르게 연결되었는가 | CT 미연결 또는 잘못된 네트 | PHY 측 CT → VCC, RJ45 측 CT → Bob Smith |
| ETH4 | ESD TVS가 RJ45 측에 배치되었는가 | WARN: ESD 보호 없음 → 서지 취약 | TVS 부품 존재 + RJ45 측 위치 확인 |
| ETH5 | TVS 캐패시턴스가 ≤5pF인가 | WARN: 신호 왜곡 | 데이터시트 Cj 값 확인 |
| ETH6 | Bob Smith 종단(75Ω+1nF)이 트랜스포머 2차 CT에 연결되었는가 | WARN: 커먼모드 노이즈 증가 | 종단 회로 존재 확인 |
| ETH7 | Bob Smith 캐패시터 정격이 ≥2kV인가 | 절연 파괴 위험 | 부품 사양 확인 |
| ETH8 | RJ45 쉴드가 섀시 GND에 연결되었는가 | WARN: EMC 성능 저하 | 쉴드 핀 네트 확인 |
| ETH9 | MDI 차동 쌍(TX±, RX±)이 모두 연결되었는가 | 미연결 → 통신 불가 | MDI 네트 4개 추적 |
| ETH10 | PHY의 REXT(기준 저항)가 데이터시트 권장값으로 연결되었는가 | REXT 누락/오류 → PHY 미동작 | 저항값 + GND 연결 확인 |
