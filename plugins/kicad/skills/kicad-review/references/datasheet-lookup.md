## 검색 전략

### 우선순위

1. **회로도 내 Datasheet URL** — `(property "Datasheet" "https://...")` 필드가 있으면 해당 URL 직접 사용
2. **제조사 공식 사이트** — `site:ti.com`, `site:amd.com` 등 제조사 도메인 한정 검색
3. **부품 유통사** — Mouser, DigiKey, LCSC 등의 부품 페이지 (데이터시트 링크 포함)
4. **일반 웹 검색** — 위 방법으로 찾지 못할 때 최후 수단

### 검색어 구성

```
기본: "{파트넘버} datasheet pdf"
제조사 한정: "{파트넘버} datasheet site:{제조사도메인}"
대안: "{파트넘버} application note"
하드웨어 설계 가이드: "{파트넘버} hardware design guide"
```

파트넘버에서 패키지/온도 접미사를 제거하면 검색 성공률이 높아진다:
- `STM32F407VGT6` → `STM32F407` 또는 `STM32F407xx`
- `XC7A200T-2FBG484I` → `XC7A200T` 또는 `Artix-7`
- `DP83848CVV/NOPB` → `DP83848`

## 검증 항목

각 IC에 대해 데이터시트를 확보한 뒤 아래 항목을 순서대로 검증한다.

### 1. 절대 최대 정격 (Absolute Maximum Ratings)

데이터시트에서 반드시 확인해야 하는 첫 번째 항목이다. 이를 초과하면 IC가 즉시 또는 점진적으로 손상된다.

| 확인 항목 | 대조 대상 |
|-----------|----------|
| 전원 전압 최대값 | 회로도의 전원 레일 전압 |
| 입력 핀 전압 범위 | 연결된 신호의 전압 레벨 |
| 출력 핀 최대 전류 | 부하의 소비 전류 |
| 동작 온도 범위 | 사용 환경 (상업용 0~70°C vs 산업용 -40~85°C) |
| ESD 등급 | 외부 커넥터 연결 여부 |

### 2. 권장 회로 (Typical Application / Reference Design)

데이터시트의 Application Circuit과 실제 회로도를 대조하여 다음을 확인한다:

- **필수 외부 부품 존재 여부**: 바이패스 캡, 부트스트랩 캡/저항, 피드백 저항, 종단 저항
- **부품값 일치**: 데이터시트 권장값 vs 회로도 실제 값
- **연결 구조 일치**: 핀 연결 순서, 접지 경로, 전원 경로

### 3. 핀별 기능 검증

| 핀 유형 | 검증 내용 |
|---------|----------|
| 전원 핀 | 모든 VCC/GND가 연결되었는지, 디커플링 캡 근접 배치 |
| 설정/스트랩 핀 | 풀업/풀다운으로 올바른 모드 선택 |
| 클럭 입력 | 주파수, 진폭, 결합 방식(AC/DC) |
| 데이터 버스 | 핀 순서, 종단, 인터페이스 모드 |
| 아날로그 핀 | 필터링, 기준 전압 |
| NC 핀 | 데이터시트에서 "must be unconnected" vs "connect to GND" 구분 |
| 출력 핀 | 구동 능력 대비 부하 적합성 |

### 4. 전원 IC 특화 검증

벅 컨버터, LDO 등 전원 IC는 추가 검증이 필요하다:

| 항목 | 계산/대조 방법 |
|------|---------------|
| 출력 전압 | Vout = Vref × (1 + R_top/R_bottom) — 피드백 저항 분배비로 계산 |
| 입력 캡 | ESR, 용량이 데이터시트 권장 범위 내 |
| 출력 캡 | ESR, 용량이 안정성 조건 충족 |
| 인덕터 | 인덕턴스 값, 포화 전류 > 최대 부하 전류 |
| 스위칭 주파수 | 데이터시트 설정 방법 (고정/가변, RT 저항) |
| 소프트 스타트 | SS 캡/저항 → 시작 시간 계산 |

### 5. PHY/트랜시버 특화 검증

| 항목 | 검증 내용 |
|------|----------|
| REXT 저항 | 1% 정밀 저항, 데이터시트 권장값 (보통 12.1kΩ 또는 2.49kΩ) |
| MDI 매칭 | TX±, RX± 차동 페어 임피던스 (100Ω) |
| 모드 스트랩 | 부팅 시 모드 결정 핀의 풀업/풀다운 값 |
| 클럭 소스 | 25MHz/50MHz 크리스탈 또는 오실레이터 |
| LED 구성 | LED 전류 제한 저항 |

## 비교 보고서

각 IC별로 아래 형식으로 검증 결과를 정리한다:

```markdown
### {IC Reference} — {파트넘버}

**데이터시트**: {URL 또는 출처}

#### 정격 검증

| 항목 | 데이터시트 기준 | 설계 값 | 판정 |
|------|---------------|---------|------|
| VCC 범위 | 2.97~3.63V | 3.3V | OK |
| 입력 전압 | -0.3~VCC+0.3V | 3.3V LVCMOS | OK |
| 동작 온도 | -40~85°C (산업용) | 실내 환경 | OK |

#### 권장 회로 대조

| 항목 | 데이터시트 권장 | 회로도 실제 | 판정 |
|------|---------------|------------|------|
| VCC 바이패스 | 100nF + 10μF | C1=100nF, C2=10μF | OK |
| REXT 저항 | 12.1kΩ ±1% | R5=12.1kΩ | OK |
| MDI 직렬 저항 | 없음 (내부 종단) | 없음 | OK |
| LED0 전류 제한 | 330Ω (10mA) | R6=330Ω | OK |
| RBIAS 저항 | 없음 | — | N/A |

#### 핀 연결 검증

| 핀 | 기능 | 데이터시트 요구 | 회로도 연결 | 판정 |
|----|------|---------------|------------|------|
| 1 | VCC | 3.3V 전원 | +3V3 네트 | OK |
| 2 | REXT | 12.1kΩ to GND | R5 to GND | OK |
| 24 | NC | Must be unconnected | 미연결 | OK |
```

## 검색 실패 시 대응

데이터시트를 찾을 수 없는 경우:

1. 파트넘버의 접미사를 변경하여 재검색 (패키지, 온도 등급)
2. 제조사의 제품 셀렉터/카탈로그 페이지 검색
3. 유통사(Mouser, DigiKey)에서 파트넘버 검색
4. 찾을 수 없으면 보고서에 명시: `"데이터시트 미확보 — 해당 IC의 검증 불가, 수동 확인 필요"`
5. 확인되지 않은 항목을 추측하지 않는다
